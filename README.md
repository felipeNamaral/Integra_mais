# Integra+

O **Integra+** é uma plataforma web desenvolvida para facilitar a conexão entre **usuários, empresas e unidades de serviço**. O sistema permite que usuários encontrem empresas, visualizem informações relevantes e salvem unidades como favoritas.

O projeto envolve **modelagem de banco de dados (MER/DER)** e o desenvolvimento de uma aplicação web para organizar e disponibilizar essas informações de forma estruturada.

## Funcionalidades

- Cadastro de usuários
- Cadastro de empresas
- Cadastro de unidades
- Busca de empresas e unidades
- Sistema de favoritos
- Gerenciamento de dados da plataforma

## Tecnologias Utilizadas

- HTML5
- CSS3
- Node.js
- Express.js

## Objetivo do Projeto

O objetivo do **Integra+** é criar uma plataforma que facilite o acesso a informações sobre empresas e unidades de atendimento, permitindo que os usuários encontrem serviços de forma simples e organizada.
